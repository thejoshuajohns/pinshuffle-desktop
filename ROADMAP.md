# Roadmap

## Now

- Stabilize the v2 workspace and package interfaces
- Expand selector contract fixtures for more Pinterest UI variants
- Improve desktop recovery flows and job inspection
- Harden CLI ergonomics around config profiles and job selection

## Next

- Bounded parallel scraping with adaptive rate limiting
- Better visual-cluster scoring and duplicate detection
- Job export/import for reproducible bug reports
- Improved diagnostics bundles for failed publish attempts

## Later

- Scheduled reshuffles
- Board comparison and diffing
- Multi-provider plugin surface beyond Pinterest
- Cloud or web control plane for remotely managed shuffle jobs
- macOS, Windows, and Linux release parity
